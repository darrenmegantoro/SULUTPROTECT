Place the following image files in this folder (public/images):

1. hero-background.jpg
   - Beranda hero background (navy overlay applied in Hero.tsx).
   - Recommended: wide landscape (e.g. 2000x900px or larger), .jpg.

2. sulut-protect-logo.png
   - SULUT PROTECT logo for navbar, login, and admin sidebar.
   - Recommended: square PNG with transparent background.

3. apis-avatar.png (optional)
   - APIS avatar in chat launcher, headers, and messages.

If logo or hero files are missing, the site still renders:
- Logo falls back to the ShieldCheck icon.
- Hero falls back to solid navy background.
