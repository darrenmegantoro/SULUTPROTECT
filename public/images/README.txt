Place the following image files in this folder (C:\Users\Darren\sulut-protect\public\images):

1. hero-background.jpg
   - Used as the Beranda hero background (with a navy overlay applied automatically).
   - Reference: the Bank Indonesia office / consumer-service photo.
   - Recommended: wide landscape (e.g. 2000x900px or larger), .jpg.
   - If this file is missing, the hero still renders with a solid navy background (no broken image).

2. apis-avatar.png
   - APIS (Asisten Pengaduan Interaktif Sulut) avatar in chat launcher, headers, and messages.
   - Recommended: square portrait, .png with transparent or dark background.

3. map-reference.jpg (OPTIONAL)
   - Only needed if you decide to swap the live Google Maps embed in the footer
     for a static map screenshot. By default the footer uses a live embed + a
     "Buka Peta" button, so this file is not required.
